3) Installing from PyPI
	The most common usage of pip is to install from the Python Package Index 
	using a requirement specifier. Generally speaking, a requirement specifier 
	is composed of a project name followed by an optional version specifier. 
	PEP 440 contains a full specification of the currently supported specifiers.
